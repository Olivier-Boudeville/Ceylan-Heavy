Ceylan's Bug parade

Despite relatively high coding standards and a rather complete non-regression test suite, we still need to pursue and eliminate bugs. You can help us in this never-ending quest by kindly submitting the ones you might discover.

Please look at our SourceForge's tracker tool to get an insight and, if needed, report a bug.

Otherwise, you can send us a bug report form, that you can generate using trunk/src/conf/generateBugReport.sh.
