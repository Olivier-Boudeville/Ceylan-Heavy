Homepage of the Ceylan 0.6 version

Overview

As of Tuesday, October 2, 2007, this is the latest Ceylan's stable version.

If you are searching for general informations about the Ceylan library (not specific to the 0.6 version), prefer referring to our Ceylan official website.

Main changes

  • The filesystem layer has been completely rewritten so that the same user code can be developed, not depending on the targeted platform (see filesystem managers)
  • Support for the Nintendo DS added (on the ARM9)
