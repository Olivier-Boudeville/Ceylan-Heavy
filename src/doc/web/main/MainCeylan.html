Welcome to the Ceylan project



The Ceylan project has for goal to provide a free portable high-level open-source general-purpose C++ library.

Ceylan has been created in order to gather all generic developments needed by the OSDL project.

The Ceylan library has now reached a certain level of stability and maturity, thanks to daily practical application and intensive testing.

The Ceylan documentation is quite sparse at the moment. Apart from the User Guide and the Developer Guide, one should preferably look at the generated API documentation, which embeds quite a lot of informations.

Finally, if a technical matter remains unclear, please drop us a line and we will add the relevant documentation section.


We hope that Ceylan might be of interest for your projects !
