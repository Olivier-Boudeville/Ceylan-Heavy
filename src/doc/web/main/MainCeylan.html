Welcome to the Ceylan project



The Ceylan project has for goal to provide a free portable high-level open-source general-purpose C++ library.

Ceylan has been created in order to gather all generic developments needed by the OSDL project.

This is an early stage of Ceylan's development, so very little information is to be found on this website. We nevertheless hope it will get bigger and bigger, as the source code grows.


We hope that Ceylan might be of interest for your projects !
