Changes for version 0.1 [latest beta version]




Ceylan version 0.1


2003, December 29
Many improvements that were not reported here, we are heading towards Ceylan 0.2 to be imported on our CVS :
  • from now on, we use GNU libtool to build our static and shared libraries
  • our makefiles allow colorization of messages, so that, when one builds Ceylan from scratch, he is not overwhelmed by messages without being able to catch what is going on
2003, July 28
Makefile system improved with new useful targets, such as checktools and stat. Use make help to know what is available.
2003, July 26
Log system improved and totally rebuilt.
2003, July 25
Some refactoring done for :
  • Exceptions,
  • Verbosity level support added,
  • GNUmakefile system improved by better compiler handling,
  • documentation files (this file, TODO etc.) converted to HTML and put in src/doc/web/main/documentation subdirectory with links in src/code in order not to forget them.
2003, April 4
Singleton class added, should be templated soon, if possible.
