Homepage of the Ceylan 0.7 version

Overview

Starting from Saturday, April 4, 2009, this is the current stable Ceylan's version.

Ceylan 0.7 has been mainly tested on 32-bit GNU/Linux hosts. Previous versions were successfully tested on Windows XP, Nintendo DS, FreeBSD, NetBSD, on various 32 and 64-bit machines.

If you are searching for general information about the Ceylan library (not specific to the 0.7 version), prefer referring to our Ceylan official website.

Main changes

Ceylan license has been extended and updated, went from pure LGPL to disjunctive bi-license LGPL/GPL. It allows developers using GPL software to use Ceylan as well.

Many features were added, including a template-based alternative MVC framework. Some bugs have been fixed. Lots of various scripts in shell and Erlang were added. WOOPER was updated.
